# Volty Packages

This directory contains framework-specific packages built on top of the core Volty CSS library.

| Package | Description | Status |
|---|---|---|
| [`@volty/react`](./react/) | React component wrappers with TypeScript | 🚧 Pre-release |

## Development

From the repo root:

```bash
# Build the core CSS library
npm run build

# Work on a specific package
cd packages/react
npm install
npm run dev
```
